#JMeter from Gradle

Start the JMeter GUI with: 
```gradlew jmeterEditor```


# JMeter plugin
See: https://github.com/kulya/jmeter-gradle-plugin/wiki

### Run JMeter test
    gradlew jmeterRun

At project directory

### Run JMeter GUI
    gradlew jmeterEditor

At project directory

### Clean the reports
    gradlew jmeterCleanReport

### List all the testPlan
    gradlew jmeterListTestPlan
